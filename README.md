## 🎯 Жоба туралы

**KidSecurity** — балаларды интернеттегі қауіптерден қорғауға арналған фронтенд веб-қосымша. Жүйе кіріс хабарламаларды автоматты түрде талдап, қауіпті мазмұн анықталған кезде ата-анаға дереу хабарлайды.

### Негізгі мақсат

- 📨 Хабарламаларды мониторинглеу және қауіпті мазмұнды анықтау
- 🚨 Автоматты ескерту жүйесі және ата-анаға хабарлау
- 📊 Балалардың онлайн белсенділігінің статистикасы
- 🌐 Толық қазақ тілді интерфейс

## ✨ Мүмкіндіктер

### 🔍 Қауіп детекциясы
- Кілт сөз негізіндегі талдау жүйесі
- 3 деңгейлі қауіп классификациясы: **Жоғары / Орта / Төмен**
- Қауіпті сөздерді хабарламада бөлектеп көрсету
- Қауіп түрлері: қорқыту, бопсалау, қысым жасау, сталкинг

### 📱 Платформалар қолдауы
- WhatsApp, Telegram, Instagram хабарламаларын мониторинглеу
- Әр платформа бойынша статистика

### 🔔 Хабарлама жүйесі
- Қауіп анықталған кезде дыбыстық дабыл (Web Audio API)
- Toast-хабарламалар (animated popup)
- Ата-анаға хабарлау симуляциясы

### 📊 Бақылау панелі
- Белсенді қауіптер саны
- Қауіпсіздік деңгейі (%)
- Деңгейлер бойынша сипаттама
- Соңғы қауіптер тізімі

### ⚙️ Жалпы мүмкіндіктер
- 🌙 Қараңғы / Жарық режим
- 📱 Толық responsive дизайн (мобиль + десктоп)
- 🔐 localStorage аутентификация
- 🔍 Хабарламаларды іздеу және сүзгілеу
- ⏱️ Жасанды real-time хабарламалар (setInterval)
- 👤 Ата-ана / Бала рөлдері

## 🚀 Орнату

### Әдіс 1 — Тікелей ашу (жергілікті)

```bash
# Репозиторийді клондаңыз
git clone https://github.com/[username]/kid-security.git

# Папкаға өтіңіз
cd kid-security

# index.html файлын браузерде ашыңыз
# (Chrome, Firefox, Edge — кез келген заманауи браузер)
```

> 💡 **Ескерту:** JSON файлдарын дұрыс оқу үшін жергілікті сервер арқылы ашу ұсынылады:
> ```bash
> # Python бар болса:
> python -m http.server 8000
> # Браузерде: http://localhost:8000
> ```

### Әдіс 2 — VS Code Live Server

1. VS Code-ты ашыңыз
2. **Live Server** плагинін орнатыңыз
3. `index.html` файлын тінтуірдің оң жақ батырмасымен басыңыз
4. **"Open with Live Server"** таңдаңыз

## 🌐 GitHub Pages Deploy

### Қадам 1 — Жаңа репозиторий жасаңыз

1. [github.com](https://github.com) сайтына кіріңіз
2. `+` → **New repository** басыңыз
3. Атын енгізіңіз: `kid-security`
4. **Public** таңдаңыз
5. **Create repository** басыңыз

### Қадам 2 — Файлдарды жүктеңіз

**Git арқылы:**
```bash
git init
git add .
git commit -m "feat: Initial KidSecurity release"
git branch -M main
git remote add origin https://github.com/[username]/kid-security.git
git push -u origin main
```

**Немесе drag & drop:** GitHub репозиторийіне файлдарды тікелей сүйреп апарыңыз.

### Қадам 3 — GitHub Pages қосыңыз

1. Репозиторий → **Settings** қойындысы
2. Сол жақ мәзірден **Pages** таңдаңыз
3. **Source** → `Deploy from a branch`
4. **Branch** → `main` → `/ (root)`
5. **Save** басыңыз

### Қадам 4 — Сайтыңыз дайын! 🎉

```
https://[username].github.io/kid-security/
```

> ⏱️ Deploy 1–3 минут алады. Actions қойындысынан күйін бақылауға болады.

## 🧠 Қауіп детекциясы

`ThreatEngine` модулі хабарламаларды мынадай алгоритм бойынша талдайды:

### Кілт сөздер (үлгі)

```javascript
// Жоғары деңгей (High)
['өлтіремін', 'зорлаймын', 'өлесің', 'жоямын', 'ренжітемін']

// Орта деңгей (Medium)  
['ақша бер', 'суреттерің бар', 'бопсалаймын', 'мектепте ұрамын']

// Төмен деңгей (Low)
['ешкімге айтпа', 'тек өзімізде', 'жасырын']
```

### Талдау нәтижесі

```javascript
ThreatEngine.analyze("Саған ақша бер деп айтамын")
// → {
//     isThreat: true,
//     severity: "medium",
//     keywords: ["ақша бер"],
//     type: "extortion"
//   }
```

### Қауіп түрлері

| Тип | Қазақша | Сипаттама |
|-----|---------|----------|
| `threat` | Қорқыту | Дене жарақаты немесе зиян келтіру |
| `extortion` | Бопсалау | Ақша немесе бағалы зат талап ету |
| `blackmail` | Шантаж | Суреттер немесе ақпаратпен қорқыту |
| `bullying` | Зорлық | Мектеп немесе онлайн зорлық |
| `stalking` | Бақылау | Жасырын бақылау белгілері |
| `harassment` | Қысым | Психологиялық қысым |
